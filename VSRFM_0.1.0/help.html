This app is a web interface for VSRFM/VSRFM-s scores calculation, to aid in variant prioritization.

1. Click on “VSRFM” or “VSRFM-s” to select the score to calculate.

2. Click on “browser" to upload variants data.

Uploaded data must be in tab separated values with ".tsv" or ".txt" extesion. For VSRFM must contain the following variables under these exact headers:

gnomAD_AF

CADD_PHRED

Condel

DANN_score

MetaLR_score

MetaSVM_score

REVEL_score

phastCons100way_vertebrate

phastCons20way_mammalian

phyloP100way_vertebrate

phyloP20way_mammalian

For VSRFMs, in addition to the above variables, must contain:

ada_score

rf_score
